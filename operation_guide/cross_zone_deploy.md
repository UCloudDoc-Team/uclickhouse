# 跨可用区部署

数据仓库 UClickHouse 支持多可用区部署。相比单可用区部署，多可用区架构具备更强的容灾能力，可有效应对机房级故障，保障服务的高可用性。

目前，UClickHouse 提供双可用区和三可用区两种部署模式1： 

- 双可用区部署：Zookeeper集群节点与Server计算节点分布在两个可用区内；
- 三可用区部署：Zookeeper集群节点与Server计算节点分布在三个可用区内。

在多可用区部署模式下，系统会将同一分片的副本节点分散在不同可用区，确保副本之间具备可用区级隔离。当某一可用区发生故障时，其他可用区内的 Server 节点和 Zookeeper 集群仍可正常提供服务，从而实现业务的连续性与稳定性。
